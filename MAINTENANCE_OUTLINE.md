# Char Phone 维护大纲

## 1. 项目定位

这是一个沉浸式 char 虚拟手机系统。用户导入角色卡后，系统生成这个角色的私人手机体验：桌面、主题、微信、QQ、电话、视频通话、日记、相册、备忘录、通讯录、查手机、预设管理等都围绕同一个 char 展开。

第一阶段优先做核心沉浸链路：

- 角色导入
- 主题系统
- 微信 / QQ 聊天
- 语音 / TTS
- 电话 / 视频通话
- 查手机
- 日记
- 关系记忆

## 2. 当前状态

当前项目是 Vite + React + Zustand 前端原型。

已具备：

- 手机壳容器和桌面
- 两页桌面 app 布局
- 可拖动布局编辑
- 图床 / 照片组件上传
- 自定义小组件
- 角色卡导入入口
- 微信 / QQ 聊天框架
- 语音消息和浏览器 TTS 播放
- 电话界面
- 视频通话界面
- 日记、备忘录、查手机框架
- 设置页
- 主题页
- 当前主题：奶油手绘、哥特玻璃

最近整理：

- 删除旧主题：微信拟真、夜间奶油、霓虹、水墨、iOS 浅色。
- 保留奶油手绘。

- 给 `.phone-shell` 加固定手机尺寸兜底，避免浏览器拉宽手机。

## 4. 模块大纲

### 4.1 桌面 / 手机壳

用途：

- 虚拟手机主入口。
- 承载 app 图标、图床、时间卡片、小组件、底部 dock。

当前状态：

- 支持两页桌面。
- 支持 app、图床、时间卡片、小组件拖动。
- 支持自动对齐 / 自由移动。
- 手机壳尺寸已固定在 390px 设计宽度附近。

重要文件：

- `src/App.tsx`
- `src/index.css`

下一步：

- 做锁屏作为第一屏。
- 桌面布局按主题分别设计。
- 加通知角标、未接电话、未读消息。

### 4.2 主题系统

用途：

- 决定整台 char 手机的视觉性格。
- 后续应支持壁纸、图标、字体、气泡、锁屏、来电界面一起切换。

当前状态：

- `ThemeType = 'pastel' | 'gothic'`
- 奶油手绘为基础主题。
- 哥特玻璃为灰黑玻璃手机风格，带红黑素材纹理。

重要文件：

- `src/store.ts`
- `src/App.tsx`
- `src/index.css`

下一步：

- 主题不要只换颜色，要换控件语言。
- 哥特主题继续细化：花体标题、磨砂玻璃卡片、红黑素材层、角色专属壁纸。
- 主题页增加预览图。

### 4.3 角色导入

用途：

- 导入酒馆卡 / 角色卡。
- 读取角色名称、头像、人设、世界书、开场白。
- 每个 char 维护独立手机数据。

当前状态：

- 支持 `.png` / `.json` 导入入口。
- 可编辑名称、人设、性格、开场白、系统提示词、世界书。

重要文件：

- `src/lib/charaParser.ts`
- `src/store.ts`
- `src/App.tsx`

下一步：

- 增加关系设定：朋友、恋人、同事、暧昧对象。
- 每个 char 独立保存主题、聊天、日记、相册、查手机内容。

### 4.4 微信 / QQ 聊天

用途：

- 微信适合日常、亲密、朋友圈、语音条。
- QQ 适合校园、群聊、熟人吐槽、空间互动。

当前状态：

- 有微信 / QQ 联系人列表。
- 有聊天房间。
- 支持文本、语音、贴纸、TTS 播放。

重要文件：

- `src/App.tsx`
- `src/store.ts`

下一步：

- 微信增加语音条转写。
- QQ 增加群聊、空间、个性签名、戳一戳。
- 聊天内容和关系记忆互相影响。

### 4.5 电话 / 视频通话

用途：

- 提供强沉浸电话体验。
- 模拟普通电话、深夜电话、未接来电、语音留言、主动来电。

当前状态：

- 电话页可选择角色。

### 4.6 日记 / 关系记忆

当前状态：

## 5. 开发规则

### 5.1 每次只做一个窗口任务

一个任务只聚焦一类事情：

- 只改主题
- 只改电话
- 只改角色导入
- 只改聊天

避免一次改太多导致上下文混乱。

### 5.2 文档不要比代码还重

推荐写法：

- 模块目的
- 当前状态
- 文件位置
- 下一步
- 已知问题

不推荐：

- 大量重复描述
- 每个 UI 细节都写成长文
- 把所有模块塞进一份巨大文档

### 5.3 AI 协作方式

给 AI 的上下文应包含：

- 当前目标
- 本次只改哪个模块
- 相关文件
- 当前已知问题
- 期望验收结果

多人协作时：

- 用清晰 commit message
- 用高质量 README
- 用模块文档，不让 AI 检索大量散文件

## 6. 下一步建议

推荐顺序：

1. 稳定主题系统：奶油和哥特都要布局不歪、边框不丢、文字不重叠。
2. 做锁屏：时间、日期、电量、壁纸、未读消息、未接电话。
3. 完善角色导入：关系设定、角色专属手机数据。
4. 做微信语音条：TTS 播放、文字转写、长按查看。
5. 做电话来电：主动来电、未接、留言。
6. 做查手机：搜索记录、备注、隐藏相册、消费记录。
