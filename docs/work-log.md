# 小手机工作记录

每次修改项目都要在这里追加记录。记录只写本次真实发生的事情，不写未来愿望。

## 记录格式

```md
## YYYY-MM-DD 标题

- 范围：改了哪些模块或文件。
- 原因：为什么要改。
- 内容：实际做了什么。
- 文档：同步了哪些文档。
- 验证：跑了哪些命令，结果如何。
- 后续：还剩什么明确事项。
```
