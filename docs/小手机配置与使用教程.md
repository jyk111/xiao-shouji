## 1. 首次进入小手机

玩家打开小手机后，会先进入登录页面。

正常流程：

1. 点击 `登录`
2. 跳转到 Discord 验证
3. 加入指定社区
4. 完成身份验证
5. 返回小手机
6. 解锁进入桌面

如果 Discord 验证暂时不可用，作者可使用隐藏后门入口。

隐藏入口：

1. 在登录页连续点击 `登录` 三次
2. 出现后门输入框
3. 输入作者提供的 6 位临时码
4. 点击 `后门进入`

后门临时码每 48 小时刷新一次。

## 2. 配置文本大模型 API

进入路径：

`桌面 -> 设置 -> 文本大模型`

推荐默认使用 DeepSeek。

### DeepSeek 配置

接口地址：

```text
https://api.deepseek.com/v1
```

API Key：

```text
填写你的 DeepSeek API Key
```

模型名：

```text
deepseek-chat
```

如果使用 `v4pro`，模型名填写：

```text
v4pro
```

小手机使用 OpenAI 兼容格式，请确认你的服务商支持：

```text
/v1/chat/completions
```

---

## 3. 拉取或手动填写模型

在 `设置 -> 文本大模型` 页面：

1. 填写接口地址
2. 填写 API Key
3. 点击 `拉取模型`
4. 选择返回的模型

如果无法拉取模型，可以手动填写模型名，例如：

```text
deepseek-chat
v4pro
gpt-4o-mini
gemini-2.5-pro
claude-sonnet-4
```

模型是否可用取决于你的 API 服务商。

## 4. 配置生图 API

进入路径：

`桌面 -> 设置 -> 生图配置`

这里用于配置 NAI 生图。微信、小红书、char 主动发图都会读取这里。

默认接口地址：

```text
https://image.novelai.net/ai/generate-image
```

API Key：

```text
填写你的 NAI API Key
```

模型：

```text
nai-diffusion-3
```

配置后可用于：

- 微信聊天 AI 生图
- 小红书封面生图
- char 主动发送图片
- AI 生图自动保存到相册

---

## 5. 配置 TTS 语音

进入路径：

`桌面 -> 设置 -> TTS 语音`

可选类型：

- 浏览器语音：无需 Key
- OpenAI TTS：需要 OpenAI API Key
- Gemini TTS：需要 Gemini API Key
- 本地 HTTP：适合 GPT-SoVITS、IndexTTS、Piper 等本地服务

开启后，可用于：

- 微信语音条
- 电话语音
- 角色语音回复

---

## 6. 微信使用说明

进入路径：

`桌面 -> 微信`

微信支持：

- 文字聊天
- 语音条
- AI 生图
- 语音通话
- 视频通话
- 朋友圈
- 照片墙
- 角色主动发图

聊天会读取：

- 当前角色设定
- 微信预设
- 最近聊天上下文
- 相册、日记、备忘录等手机数据

---

## 7. 预设使用说明

进入路径：

`桌面第二页 -> 预设`

每个软件都有独立预设，例如：

- 微信 / QQ
- 浏览器
- 小红书
- B站
- 电话
- 音乐
- 小剧场
- char 主动

预设条目可设置发送身份：

```text
system
user
assistant / model
```

建议：

- 规则、世界观、限制：使用 `system`
- 玩家要求：使用 `user`
- 示例回复：使用 `assistant / model`

---

## 8. 相册使用说明

进入路径：

`桌面 -> 相册`

相册支持：

- 上传照片
- 添加标签
- 按来源分类
- 微信图片联动
- 小红书封面联动
- 锁屏壁纸联动
- AI 生图保存

推荐标签：

```text
角色
聊天
风景
头像
锁屏
小红书
AI生图
```

---

## 9. 更换锁屏壁纸

进入路径：

`桌面 -> 相册 -> 打开照片`

选择：

```text
设为锁屏
```

如果玩家没有设置锁屏壁纸，小手机会使用当前主题的默认锁屏风格。

## 11. 数据备份与导入

进入路径：

`桌面第二页 -> 数据备份`

用于：

- 导出当前数据
- 更新 APK 后导入旧数据
- 保留角色、聊天、相册、预设、主题等内容

建议每次更新 APK 前先导出备份。

---

## 12. 玩家最简流程

玩家只需要：

1. 安装 APK
2. 打开小手机
3. 点击登录
4. 完成 Discord 验证
5. 解锁进入桌面
6. 进入设置配置 API
7. 导入角色
8. 打开微信开始聊天

只配置文本大模型也可以正常使用。生图和 TTS 是增强功能。
