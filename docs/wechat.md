# 微信模块

## 当前状态

- 微信入口位于桌面 app，内部保留 4 个底部页签：微信、通讯录、发现、我。
- 微信聊天列表和通讯录都直接读取全局 `characters`，导入角色卡后会同步出现在微信聊天列表和微信通讯录。
- 聊天主界面支持文字消息、图片消息、带描述的表情包消息、语音条消息、浏览器 TTS 播放、长按/右键/按钮查看语音转写。
- 通讯录保留新的朋友、群聊、标签；群聊可选择联系人创建，标签可给联系人填写。
- 发现页包含朋友圈、照片墙和内置表情包库；表情包可上传并编辑文字描述。
- 我页面支持修改头像、昵称、状态、微信号；默认微信号为 `9142`，状态输入移到列表行里，避免覆盖个人资料区。
- 微信样式跟随全局主题：奶油主题使用浅色奶油手绘微信风，哥特主题使用暗色微信风。

- 假语音输入框占位字缩小并固定一行。
- 消息操作改为点击/长按气泡后呼出，不再每条常驻；用户消息只保留引用和撤回，char 消息可引用、复制、收藏、撤回或删除。
- 消息补充发送时间；char 回复时显示“正在输入中”，并按延迟一条条出现。
- 输入栏右侧收紧为加号面板，表情、图片、语音通话、视频通话统一从加号展开。
- 群聊会同步出现在微信聊天列表，通讯录里可点开群聊，也可解散群聊。

- 微信聊天预设支持导入 `.json/.txt` 或粘贴酒馆式预设文本，能映射 name/prompt/temperature/depth/max_tokens 等字段。
- 微信底部页签红点改为只显示未读数，不再因为存在历史会话一直显示。
- 用户消息操作按钮区分为绿色“引用”和红色“撤回”，不再两个箭头长得像。
- 输入栏左侧语音按钮删除；语音条、表情、图片、语音通话、视频通话都收进发送旁边的加号面板。
- 表情包描述不再显示在聊天气泡里，只作为“表情包注释”进入 AI 上下文。

## 重要文件

- `src/App.tsx`：微信外壳、聊天主界面、语音条/转写/TTS。
- `src/wechat/chats/WeChatChats.tsx`：微信聊天列表。
- `src/wechat/contacts/WeChatContacts.tsx`：微信通讯录、导入角色卡、群聊、标签。
- `src/wechat/discover/WeChatDiscover.tsx`：发现页、朋友圈、照片墙、表情包。
- `src/wechat/me/WeChatMe.tsx`：我页、收藏、订单与卡包、微信聊天预设设置。
- `src/wechat/shared/WeChatShared.tsx`：微信页签共享 UI 和格式化/预设工具。
- `src/wechat/ai/`：微信 AI 活人感提示词、生活动作解析和新手教程。
- `src/wechat/stickers/`：emoji/sticker 风格表情包 starter manifest 和素材来源说明。
- `src/store.ts`：微信个人资料字段、照片墙、朋友圈、表情包描述、群聊、联系人标签、聊天记录、旧数据迁移。
- `src/index.css`：微信浅色/暗色主题、聊天气泡、底部页签、滚动条隐藏。
- `src/lib/charaParser.ts`：酒馆 PNG/JSON 角色卡解析。
- `src/lib/utils.ts`：兼容 ID 生成。
